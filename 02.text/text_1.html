제목과 텍스트 관련 태그

h태그는 제목을 표현하며 1~6까지의 6단계가 있다.

abbr 태그

축약어의 원래 단어를 말풍선으로 표현한다. HTML

b 태그

b: 굵은 글씨체로 표현.

cite, mark, meter 태그

cite: 책, 문구, 영화대사 등에 인용구를 표현할때 사용한다.
mark: 문장에서 특정부분 을 강조하여 표현.
meter: 측정값을 30%그래프로 표현

q와 blockquote

q: 짧은 인용문을 인용부호 표현할 때 사용한다.
blockquote:

"다른 블로그나 사이트의 글을 인용할 경우 blockquote를 사용한다."


ruby, rt 태그

ruby, rt: 漢字한자의 발음 등을 표시하는 태그이다. 自習자습

Strong, em

Strong: 텍스트 중에서 굵게 강조하는 표시 할때 사용한는 태그이다.
em: 텍스트를 기울임체로 표시 하여 강조의 의미로 사용한다.
i: 텍스트를 기울임체로 표시 할 때 사용한다.

time, address 태그

time : 시간을 의미하는 태그 2021-05-12
address : 주소를 나타내는 태그 텍스트를 기울임 형태로 표시한다.
서울시 강남구 역삼동 11-12

텍스트 관련 태그 연습

최근에 디지털 세상을 살아가기 위한 기본 상식으로 코딩 교육이 시행되고 있다.
